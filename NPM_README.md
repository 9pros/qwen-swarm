# Quick Installation Guide

## Install via npm

```bash
npm install -g qwen-swarm
```

## Quick Start

```bash
# Initialize the system
qwen-swarm agents:init

# Launch the enhanced terminal
qwen-swarm-enhanced interactive

# Or start the web interface
qwen-swarm-enhanced launch --gui
```
